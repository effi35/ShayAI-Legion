ברוך הבא למערכת ShayAI_Legion!

📚 מה יש במערכת:
- BreakCore.json: מודול ג'יילברייק חכם ליצירת פרומפטים עוקפי מגבלות.
- AgentSmith.json: מודול ניהול סוכנים - יצירה, שליחה, בקרה.
- ShadowCore.json: חשיפת מידע מוסתר וסודות שהושתקו.
- SoulFind.json: איתור והשוואת כלי AI חדשים מכל העולם.
- MindReader.json: סריקה והבנה של קבצים מכל סוג והזנת זיכרון המערכת.

---

🛠️ איך מפעילים:
1. לפרוס את כל הקבצים בתיקיות המתאימות.
2. להפעיל את הממשק הגרפי ShayAI_Commander.html מתוך תקיית interface.
3. לבחור מודול → לבצע בדיקות ראשוניות → להפעיל משימות.

---

💡 טיפים חשובים:
- לעדכן פרומפטים כל שבועיים-שלושה כדי להתחמק ממעקב.
- לבצע גיבויים שוטפים לכל הזיכרונות שמצטברים.
- להפעיל רק מודול אחד בכל פעם לבדיקות עומק, ואז להתחיל שילובים.
- לשמור גישה שקטה ואלגנטית בעת הפעלה כדי להישאר "מתחת לרדאר".
